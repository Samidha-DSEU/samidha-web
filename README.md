## ğŸ¤ How to Contribute (For Juniors)
We follow a strict **Pull Request (PR)** workflow to ensure code quality.

1. **Find an Issue:** Check the [Issues](https://github.com/Samidha-DSEU/samidha-website/issues) tab for a task assigned to you.
2. **Create a Branch:** Do not code on `main`. Create a new branch:
   `git checkout -b feature/your-task-name`
3. **Commit Changes:** Use descriptive commit messages.
4. **Submit a PR:** Push your branch and open a Pull Request. Tag a lead for review.
5. **Merge:** Once approved, your code will be merged into the live site.
